# Job Description - Full-Stack (Java)

## Role Requirements

Developer with these technical skills:

- JRE, JDK
- Web Frameworks: Spring Boot, Jakarta EE
- Database (ORM): Hibernate, Spring Data JPA
- Build Tools: Maven, Gradle
- Testing: JUnit, Mockito
- Cloud/Microservices: Quarkus

## Key Responsibilities

- Design and develop scalable applications using Java technologies
- Implement microservices architecture
- Work with cloud platforms (AWS preferred)
- Write unit and integration tests
- Collaborate with cross-functional teams

## Technical Stack

- Java/Spring Boot
- Quarkus
- Hibernate/Spring Data JPA
- Maven/Gradle
- JUnit/Mockito
- AWS/Cloud
