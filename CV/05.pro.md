## register-domain-events-jpa

**Full name:**: 
    - jcarloshg/register-domain-events-jpa

**Summary:**
    - A Spring Boot application for registering and retrieving domain events. It provides RESTful endpoints to create and list events, storing them in a database. The project demonstrates Domain Driven Design, clean architecture, validation, exception handling, and supports both PostgreSQL and H2 databases.
