# Job Description - Frontend Developer

## Responsibilities

a) Develop reusable functionalities and components in frontend.

b) Unit testing of own developments.

d) Code versioning.

## Requirements and Qualifications

- Advanced/Conversational English
- Proven experience as a front-end developer or similar role.
- Proficient in HTML, NextJS, React, JavaScript, TypeScript, API usage
- Consume APIs using at least "fetch".
- Create pages and display results.
- Know how to add interactivity with buttons, and handle success and error messages.
- Experience with front-end frameworks such as React, Angular, or Vue.js.
- Familiarity with version control systems (e.g., Git)
- Strong understanding of responsive design principles.
- Knowledge of SEO principles and best practices.
- Excellent communication and problem-solving skills.
- Ability to work independently and as part of a team.
- Attention to detail and passion for creating visually attractive and easy-to-use interfaces.
- HCL Commerce (desirable)

## Preferred Qualifications

- Experience with UI/UX design.
- Knowledge of back-end technologies such as Node.js or PHP.
- Familiarity with agile development methodologies.
