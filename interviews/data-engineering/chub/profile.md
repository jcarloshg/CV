Job Spec – MLOps / AI Engineer

Buscamos un profesional para implementar pipelines de MLOps, automatizando el ciclo de vida de modelos de IA: entrenamiento, validación, despliegue, monitoreo y reentrenamiento. El candidato integrará herramientas de versionamiento de modelos, CI/CD y observabilidad para garantizar operación continua y confiable de las soluciones en producción.

El perfil debe ser capaz de colaborar con equipos de Data Engineering, Cloud, AI/ML y DevOps, garantizando la calidad, escalabilidad y eficiencia de los sistemas de inteligencia artificial.

Responsabilidades principales
• Implementación y mantenimiento de pipelines MLOps para modelos de IA.
• Gestión de versiones de modelos, integración con CI/CD y monitoreo en producción.
• Desarrollo y despliegue de APIs y microservicios para soluciones AI.
• Integración de plataformas cloud y herramientas de orquestación de ML.
• Colaboración con equipos de datos para construcción de pipelines ETL/ELT.
• Aplicación de principios de Responsible AI y AI Ethics, detección de sesgos y privacidad.
• Visualización y análisis de datos para monitoreo de modelos.

Stack tecnológico

Frameworks y ML/DL
• TensorFlow, PyTorch, Scikit-learn, Keras, Hugging Face

Lenguajes
• Python, R, PowerShell, JavaScript, SQL, Java, C#

MLOps y Deployment
• MLflow, Kubeflow, Docker, Kubernetes, CI/CD para ML

Cloud Platforms
• Azure ML, AWS SageMaker, Google Cloud AI Platform

Data Engineering
• ETL/ELT, Spark, Databricks, Data Pipelines

LLM & Generative AI
• GPT, BERT, fine-tuning, prompt engineering, RAG

Computer Vision

NLP
• Tokenización, embeddings, análisis de sentimiento, named entity recognition

Estadística y Matemáticas
• Probabilidad, estadística inferencial, álgebra lineal, cálculo

Visualización de Datos
• Matplotlib, Seaborn, Plotly, Power BI, Tableau

Versionamiento de Código
• Git, GitHub/GitLab/Azure DevOps

APIs y Microservicios
• REST APIs, FastAPI, Flask, arquitectura de microservicios

Habilidades deseables
• Colaboración en equipos multidisciplinarios.
• Capacidad de documentación técnica y buenas prácticas de desarrollo.
• Enfoque en escalabilidad y performance de modelos en producción.
• Conocimiento de mejores prácticas de Responsible AI.
