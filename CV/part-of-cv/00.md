
# USA Resume (ATS) + Application Cheat Sheet

## 1) Formato ATS (reglas rápidas)

- **Diseño simple:** sin columnas, tablas, íconos, gráficos, ni elementos decorativos.
- **Fuentes estándar:** Arial/Calibri/Times (11–12).
- **Archivo:** PDF o DOCX.
- **Encabezados estándar:** `Professional Summary`, `Technical Skills`, `Experience`, `Projects`, `Education`, `Certifications`.
- **Contacto:** siempre en el cuerpo del documento (evita header/footer).
- **Keywords:** replica palabras clave exactas del job description (skills, tools, metodologías).
- **Orden recomendado:** combinado/cronológico inverso (skills arriba + experiencia en orden inverso).

## 2) USA Resume (estructura recomendada)

1. **Header**
	- Nombre + Email + Teléfono (+52) + LinkedIn + GitHub
	- Ubicación: `City, Country` (sin dirección completa)
	- **No foto / no edad / no estado civil / no fecha de nacimiento**
	- **Formato de fechas:** usar formato USA `MM/YYYY` o `Month YYYY` (ej: `01/2023` o `January 2023`)
2. **Professional Summary (3–4 líneas)**
	- Rol + años + especialidad + 1–2 logros medibles + objetivo.
3. **Technical Skills** (arriba para ATS)
	- Agrupa: `Languages`, `Frameworks`, `Databases`, `Cloud/DevOps`, `Architecture`, `Testing/Tools`.
4. **Professional Experience** (cronológico inverso)
	- Bullets orientados a **impacto** (métricas) y verbos fuertes: *Architected, Led, Optimized, Delivered*.
5. **Projects (3–5 destacados)**
	- Qué resuelve + stack + link + impacto (si aplica).
6. **Education** (al final si eres senior)
7. **Optional:** `Certifications`, `Languages`, `Open Source`/community

## 3) Estrategia para aplicar (sube tu probabilidad)

- **No solo “Apply”**: identifica reclutador/hiring manager en LinkedIn y envía mensaje corto + link a GitHub/portfolio.
- **Personaliza por vacante:** resume + (cover letter opcional) con keywords reales del puesto.
- **Networking:** meetups, comunidades dev, open source, foros.
- **Follow-up:** a los 7–14 días.
- **Track:** spreadsheet con empresa, rol, fecha, contacto, estado, follow-up.
- **Work authorization:** sé honesto; considera roles remotos y empresas con sponsorship.
- **Salary:** investiga en Glassdoor / Levels.fyi / PayScale; responde con rango o `Negotiable / Market rate` si conviene.
