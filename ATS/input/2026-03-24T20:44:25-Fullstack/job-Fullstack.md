# Job Description - Fullstack (React, Python)

## Requirements:
- +5 años desarrollando aplicaciones web productivas
- Experiencia fullstack en entornos data-driven
- JavaScript/TypeScript + Python
- React, Next.js, Vue, Flask o FastAPI
- Data visualization (D3.js, Plotly, Vega-Lite, ECharts)
- Desarrollo de APIs REST o GraphQL
- SQL avanzado y modelado de datos
- Bases de datos relacionales y Redis
- Seguridad en APIs (OAuth, JWT)
- Experiencia en AWS (ECS, EKS, Lambda, S3, RDS)
- CI/CD e infraestructura como código

## Plus:
- Integración de modelos ML/AI
- Experiencia con Databricks o Snowflake
- Conocimientos en Go, Rust o Java
- Experiencia con datos científicos
